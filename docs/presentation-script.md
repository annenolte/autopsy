# Autopsy — 1-Minute Presentation Script Outline

**Setting**: You're standing in front of judges, website is on screen behind you. You'll scroll through the site as you talk. After this 1-minute pitch, you transition into the live demo.

---

## [0:00–0:10] THE HOOK (Hero section on screen)

> "Forty-one percent of developers use AI coding tools every day. They accept code suggestions in seconds — but here's the thing: nobody's checking if that code is actually safe."

*Pause for effect. Let the hero animation play.*

## [0:10–0:20] THE PROBLEM (Scroll to Problem section)

> "AI tools like Copilot and Claude Code generate code 3x faster — but studies show up to 40% of AI-generated code contains security vulnerabilities. SQL injection, XSS, hardcoded secrets — all accepted with a single Tab key. The faster you ship, the less you review."

*Gesture toward the stats cards as they animate in.*

## [0:20–0:30] THE CONSUMER (Scroll to Who It's For)

> "This affects every developer using AI tools, every engineering team shipping faster than they can review, and every security-conscious organization that needs more than grep-based scanning."

*Quick scroll — don't linger here, judges can read the cards.*

## [0:30–0:45] THE SOLUTION (Scroll to Solution/Three Modes)

> "Autopsy is the AI Vulnerability Detective. It has three modes: Scan This detects AI-generated code and flags 9 categories of vulnerabilities. Debug This traces bugs across your entire dependency graph — not just the file that crashed. Orient Me maps any codebase in seconds."

> "The key differentiator: Autopsy doesn't wait for you to ask. It builds a dependency graph of your code with Tree-sitter and NetworkX, uses a two-tier Claude pipeline — Haiku for fast triage, Sonnet for deep reasoning — and streams results in real time."

*Scroll past Pipeline section as you say this.*

## [0:45–0:55] THE IMPACT (Scroll to Impact section)

> "92% of developers now use AI tools. Billions are lost annually to exploitable bugs. Autopsy is the missing security layer — it creates the trust that AI coding tools don't provide."

*Let the impact numbers animate in. Pause on the callout: "AI tools create code. Autopsy creates trust."*

## [0:55–1:00] TRANSITION TO DEMO

> "That's the pitch. Now let me show you it working live."

*Open VS Code or terminal for the live demo.*

---

## Tips for delivery

- **Don't read the website** — the judges can read. You're narrating the story, not the text.
- **Maintain eye contact** with judges, not the screen.
- **Scroll smoothly** — assign a teammate to scroll if possible, so your hands are free.
- **Hit the three pillars**: Problem (AI code is unsafe) → Consumer (everyone using AI tools) → Solution (Autopsy catches what humans miss).
- **The "trust" line is your closer** before the demo. Land it with confidence.
- **Practice the timing** — 60 seconds goes fast. Cut words, not ideas.
